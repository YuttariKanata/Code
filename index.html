1. 高速積分 (Julia)

Double Exponential（双曲正接）公式を使った高精度積分の例です。

julia 

using DoubleExponentialFormulas

function f(x)
    exp(-x^2)
end

result = integral(f, -1, 1)
println(result)

2. エラトステネスの篩 (C)

整数の素数判定に使われる古典的なアルゴリズムです。

c 

#include <stdio.h>
#include <stdbool.h>

void sieve(int n) {
    bool prime[n+1];
    for (int i = 0; i <= n; i++) prime[i] = true;
    prime[0] = prime[1] = false;
    for (int i = 2; i*i <= n; i++) {
        if (prime[i]) {
            for (int j = i*i; j <= n; j += i)
                prime[j] = false;
        }
    }
    for (int i = 2; i <= n; i++)
        if (prime[i]) printf("%d ", i);
}
